# 2. Problem Validation Summary

This section summarizes the research basis for the product. Full citations and sources are maintained separately in the team's research log; this is a condensed reference for engineering and design decisions.

## 2.1 Isolation & Loneliness

- 9.2 million Filipino senior citizens nationwide; majority live with extended family but report high loneliness
- Only 35% of seniors attend social activities at least once a month
- 90% have a social media account, but only 6% have regular internet access — confirms the need for assisted-use design
- TANDA-NAN field research (Manggahan Residences, Pasig) confirms seniors want to be co-creators, not charity recipients, and value social bonding tied to economic agency

## 2.2 Online Scams & Fraud

- 76% of the Silent Generation (oldest age cohort) report the highest scam exposure rate of any age group in the Philippines
- Only 38% of seniors are "basic" digital users (2024 DICT report); most ignore privacy warnings
- CICC officials confirm isolation and loneliness directly increase susceptibility to romance and investment scams — validating the combined approach
- Common scam types: phishing, OTP theft, investment/Ponzi schemes, romance scams, grandparent/"apo" impersonation scams, fake NCSC pension messages

## 2.3 Existing Solution Gaps

| Gap | Description | How Gabay Sr. Addresses It |
|---|---|---|
| No real-time protection | PH programs are education-only, reactive after the fact | Live alert system to Trusted Circle |
| No family bridge | No PH tool loops in remote family (e.g. OFWs) | Trusted Circle dashboard + summaries |
| English-only AI tools | Global tools (Hiya, SeniorShield) don't support Filipino/Taglish | Gemini API prompted for Taglish scam patterns |
| No barangay-level deployment | Existing tools are Metro Manila or corporate CSR-only | Built for OSCA/barangay rollout in Laguna |
